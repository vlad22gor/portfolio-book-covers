# Lessons

- Постоянную документацию по сайту храним в `docs/`; в `tasks/` оставляем рабочие заметки, `lessons` и `logs`.
- Сайт-портфолио строим как `Astro + TypeScript + Tailwind` с приоритетом на static-first подачу.
- `Motion` — primary runtime-motion layer для интерактивных и stateful UI-компонентов; заранее отрендеренные `webm/mp4` остаются art-directed media-слоем.
- Главная строится как 2D-галерея кейсов с точечным motion-акцентом; страницы кейсов опираются на сильную статику и контролируемый motion-layer.
- Runtime 3D-viewer не является активным baseline; знания о нём сохраняем только как архивный контекст.
- Новые интерактивные компоненты не считаются готовыми без dev-сценария в `/dev/components`.
